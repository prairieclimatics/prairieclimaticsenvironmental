# Prairie Climatics Environmental

## Professional Soil & Water Testing for Saskatchewan

Welcome to the Prairie Climatics Environmental website repository. This site showcases our environmental sampling services for farmers and agricultural operations in Saskatchewan.

### Services Offered

- **Soil Sampling & Analysis** - Comprehensive soil testing to optimize fertilizer application and maximize crop yields
- **Water Testing** - Professional water quality analysis for wells, livestock drinking water, and monitoring wells
- **Heavy Metal Analysis** - Specialized testing to detect contaminants in water supplies
- **Monitoring Wells** - Ongoing water quality monitoring for environmental compliance

### Website Features

- Responsive design for mobile and desktop
- Service descriptions and overview
- Contact form for inquiries
- GitHub Pages deployment ready

### Deployment

This website is deployed via GitHub Pages. To enable it:

1. Go to **Settings** → **Pages**
2. Select **main** branch as the source
3. Your site will be available at `https://prairieclimatics.github.io/prairieclimaticsenvironmental/`

### Getting Started

To customize this website:

1. Update contact information in `index.html`
2. Modify content in the service cards and about section
3. Update colors and styling in `styles.css` as needed
4. Add more interactivity in `script.js`

### Local Development

Simply open `index.html` in your browser to view the site locally.
